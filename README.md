## Description

Seleneko supports your Selenium development.
ex. await for accessing any site, handle any browsers, ...

| Requirement          |
| -------------------- |
| Python >= 3.9, < 4.0 |
| selenium  = 4.35.0   |

## Install

```bash
pip install git+https://github.com/ardnico/seleneko
```

- For development

```bash
git clone https://github.com/ユーザー名/プロジェクト名.git
cd プロジェクト名
pip install -e ".[dev]"
```
